init - создает репозиторий
status - проверяет его статус
    Untracked files - неотслеживаемый
    Changes to be committed - файл в индексе
    Changes not staged for commit - файл в индексе, но какие-то изменения не в индексе
add <название файла или директории> - добавить в индексе
commit -m "<ваше сообщение про коммит>" - сохранение кода, коммит
log - посмотреть журнал коммитов
remote - удаленнный репозиторий
remote add - добавить удаленный репозиторий
push - отправить на удаленный репозиторий
pull - обновить репозиторий

clone - скачать репозиторий
